## Case Study: Finding the Root of a Quadratic Function
The Newton-Raphson method was used to find the root of the quadratic function f(x) = x^2 - 2. The initial guess was 1.0, and the tolerance was set to 1e-5. The method converged to the root in 5 iterations.
## Results
The root of the function was found to be 1.4142135623730951, which is accurate to 5 decimal places.
